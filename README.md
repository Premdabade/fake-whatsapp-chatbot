# fake-whatsapp-chatbot
# 📱 Fake WhatsApp Chatbot Backend  This project is a simple backend simulation of a WhatsApp chatbot built using Java and Spring Boot.  ## 🚀 Features - REST API endpoint: `/webhook` - Accepts JSON input (simulating WhatsApp messages) - Returns predefined responses:   - Hi → Hello   - Bye → Goodbye - Logs all incoming messages in console  ##
